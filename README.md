# ✨ AIZA — Premium Restaurant Website

> AIZA — современный веб-сайт премиального ресторана с элегантным дизайном, интерактивным меню и полноценной корзиной заказов.  
> Проект создан на React + Vite и полностью готов к деплою на GitHub Pages.

---

## 🌟 О проекте

**AIZA** — это стильный ресторанный сайт с атмосферой премиального европейского заведения.  
Тёмная тема с золотыми акцентами, плавные анимации и аккуратная типографика создают ощущение высокого уровня сервиса.

Проект полностью работает на клиентской стороне — без бэкенда.

---

## 🎯 Основной функционал

### 🏠 Главная страница
- Hero-блок с атмосферным фоновым изображением
- Название ресторана AIZA
- Краткое описание
- Кнопка перехода в меню
- Блок «О нас»
- Популярные блюда

### 🍽 Меню

Категории:
- Закуски  
- Основные блюда  
- Десерты  
- Напитки  

Каждое блюдо содержит:
- Качественное изображение
- Название
- Краткое описание
- Цена в сомах (KGS)
- Кнопка «Добавить в корзину»

Всего: **17 позиций**

### 🛒 Корзина
- Добавление блюд
- Изменение количества
- Удаление позиции
- Автоматический пересчёт итоговой суммы
- Отображение общей стоимости в сомах

## 🛠 Технологический стек

| Технология | Назначение |
|------------|------------|
| React 18 | Построение интерфейса |
| Vite 5 | Быстрая сборка и dev-сервер |
| React Router v6 | Клиентская маршрутизация |
| Context API | Управление состоянием корзины |
| CSS Modules | Локальная стилизация компонентов |
| Unsplash | Качественные изображения |
